Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @ankitverma/number-formatter`

## Usage

    var numFormatter = require('@ankitverma/number-formatter');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`
